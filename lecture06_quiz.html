Q1ポートフォリオ最適化の 3 要素として正しい組み合わせは？
A資産・期待リターン・制約
B株価・出来高・配当
C銀行・証券会社・顧客
D日経平均・TOPIX・ダウ
正解: A
ポートフォリオ最適化は「どの資産にいくら配分するか」を、資産（種類）・期待リターン（各資産の収益率）・制約（合計・上限・下限）の 3 要素で組み立てます。
Q2ポートフォリオ最適化は数学的にどの種類の問題に分類される？
A非線形最適化
B線形最適化（線形計画問題）
C整数計画問題
D動的計画問題
正解: B
目的関数 $\sum w_i \mu_i$ と制約はすべて 1 次式のため、線形計画問題（LP）に分類されます。Excel の「シンプレックス LP」がそのまま使えます。
Q3今回の例題で「外貨建て（米国株 + 新興国株）≤ 50 万円」という制約を入れる経済的な理由は？
A米国株の方が手数料が高いから
B日本株を増やしたいから
C為替リスクを上限で抑えるため
D新興国株の流動性が低いから
正解: C
外貨建て資産は為替変動で円換算価値が大きく動きます。為替リスクを「外貨建て総額の上限」というハード制約で抑えるのが今回の設計です。
Q4この問題の決定変数は何個？
A3 個
B10 個
C6 個
D5 個
正解: D
日本株・米国株・新興国株・国内債券・現金の 5 資産それぞれの配分額が決定変数なので、合計 5 個です。L5 輸送問題（2 工場×3 倉庫=6 変数）よりシンプルな構造です。
Q5今回の例題で「合計 = 100 万円」という制約は数学的にどう書く？
A$\sum_{i=1}^{5} w_i = 100$
B$\sum_{i=1}^{5} w_i \leq 100$
C$\sum_{i=1}^{5} w_i \geq 100$
D$w_1 + w_2 = 100$
正解: A
5 資産の配分額をすべて足して **ちょうど 100 万円**になるよう、等号 $=$ で制約します。式は $\sum_{i=1}^{5} w_i = 100$ です。

Q1AI に問題を伝えるとき、5 資産の情報を最も正確に渡せる形式は？
A長い箇条書きで全部書く
BMarkdown 表（資産名・期待リターン・制約を列で整理）
C画像で渡す
D口頭で伝える（音声入力）
正解: B
Markdown 表は AI にとって「列の意味」が明確で、5 資産の情報を構造的に渡せます。学生が動画で写経しやすい点でも推奨です。
Q2AI が出した配分案を信用する前に、必ずすべき検算は？
A配分が偶数になっているか
BAI に「正しいですか？」と聞く
C配分の合計が 100 万円になっているか
D何もしない（AI は正確だから）
正解: C
最低限「合計 = 100 万円」「各制約を満たすか」「期待リターン $\sum w_i \mu_i$ が正しいか」の 3 点を必ず手計算で検算します。
Q3AI 回答でよくあるハルシネーション（誤回答）のパターンは？
A小数点以下を四捨五入する
B配分を全部 0 にする
C同じ答えを 2 回返す
D制約を 1 つ無視して、その分リターンを高く見せる
正解: D
AI が「外貨建て ≤ 50 万円」のような制約を無視して、米国株を 60 万円に振り、見かけ上のリターンを大きく見せることがあります。**全制約の遵守を 1 つずつ確認**するのが対策です。
Q4期待リターンを検算する数式として正しいのは？
A$\sum_{i=1}^{5} w_i \cdot \mu_i$（配分額と各資産リターン率の積和）
B$\sum w_i + \sum \mu_i$（配分額と期待リターンの単純合計）
C$\max(\mu_i)$（最大リターンの資産だけ採用）
D$\dfrac{\sum w_i}{\sum \mu_i}$（配分合計 ÷ リターン合計）
正解: A
ポートフォリオ全体の期待リターンは $\sum_{i=1}^{5} w_i \mu_i$ で計算します。配分額（万円）× リターン率（%）を全資産で合計します。Excel では SUMPRODUCT 1 セルで書けます。
Q5AI に問題を投げる前に、必ず明示すべき情報として最も重要なのは？
A自分の年齢と職業
B最大化する量（目的関数）と全制約
C使う AI のバージョン番号
D希望する計算時間
正解: B
「何を最大化（または最小化）したいか」と「すべての制約」を明確に渡さないと、AI は別の問題を解いてしまいます。Markdown 表 + 目的関数の文章説明の組み合わせが最強です。

Q15 資産の期待リターンを 1 セルで計算する Excel 関数は？
ASUM
BAVERAGE
CSUMPRODUCT
DCOUNTIF
正解: C
SUMPRODUCT は「配分額の範囲」と「リターン率の範囲」を渡すと、各行で掛け算してから合計する関数です。$\sum w_i \mu_i$ を 1 セルで書けます。
Q2Excel Solver で「目的セル」に指定するのは？
A配分額を入れる 5 つのセル
B資産名を書いたセル
C制約条件を書いたセル
D期待リターンを計算したセル（SUMPRODUCT の結果）
正解: D
目的セルは「最大化（または最小化）したい量を計算したセル」を指定します。今回は期待リターン $\sum w_i \mu_i$ を計算した SUMPRODUCT セルです。
Q3今回の問題は最大化なので、Solver の「目標」設定は？
A最大値
B最小値
C指定値
D差を最小化
正解: A
期待リターンを「**最大化**」したいので「最大値」を選びます。L5 輸送問題（コスト最小化）とは反対の選択になることに注意。
Q4今回の問題は線形最適化なので、解法は？
A何を選んでも結果は同じ
Bシンプレックス LP
Cエボリューショナリー
DGRG 非線形
正解: B
目的関数も制約も 1 次式（線形）なので、専用の高速ソルバー「シンプレックス LP」を選びます。GRG 非線形だと無駄に重く、不安定です。
Q5「制約条件の追加」で「合計 = 100 万円」を入れる方法として正しいのは？
A配分セル 1 個 = 100 を 5 回追加
BSolver では合計制約は書けない
C配分セル範囲の SUM を計算するセルを作り、それが 100 と等しい制約を追加
D不等号 $\leq$ で 100 にすればよい
正解: C
配分セルの合計を SUM で別セルに出し、「そのセル = 100」を制約として追加します。等号制約は Solver の左辺・右辺・関係（=）で明示します。

Q1PuLP で 5 資産の決定変数を 1 行で作るコードは？
Aw = np.zeros(5)
Bw = [0, 0, 0, 0, 0]
Cw = pulp.LpVariable("w")
Dw = pulp.LpVariable.dicts("w", assets, lowBound=0)
正解: D
LpVariable.dicts は辞書として複数変数を一気に作る便利関数です。lowBound=0 で「空売り不可（$w_i \geq 0$）」も同時に指定できます。
Q2目的関数「期待リターン最大化」を PuLP で書くと？
Aproblem += pulp.lpSum(w[i] * mu[i] for i in assets)
Bproblem += sum(w) + sum(mu)
Cproblem += max(mu)
Dproblem += pulp.lpSum(mu)
正解: A
lpSum と for 句を組み合わせて $\sum w_i \mu_i$ を表現します。問題に += で代入することで「目的関数として登録」されます。今回は最大化なので LpMaximize を指定して問題を作ります。
Q3PuLP の標準ソルバー（外部インストール不要で使える）は？
AGurobi
BCBC
CCPLEX
DGLPK
正解: B
CBC（COIN-OR Branch and Cut）は無料の LP/MILP ソルバーで、`pip install pulp` するだけで一緒に入ります。学習用・小規模問題には十分な性能です。
Q4制約「外貨建て（米国 + 新興国）≤ 50 万円」を PuLP で書くと？
Aproblem += w["us"] + w["em"] == 50
Bproblem += w["us"] <= 50 and w["em"] <= 50
Cproblem += w["us"] + w["em"] <= 50
Dproblem += max(w["us"], w["em"]) <= 50
正解: C
$w_{us} + w_{em} \leq 50$ をそのまま Python の式として書きます。`problem += 式` で制約を追加します。
Q5Python で得た最適解と Excel で得た最適解が一致するか確認する方法は？
APython が正しいから Excel は確認不要
B小数点 5 桁まで完全一致しない限り無効
CExcel が正しいから Python は確認不要
D配分額と期待リターンを両ツールで並べて比較する
正解: D
3 ツール（AI / Excel / Python）で同じ最適解が出ることが、解の信頼性の最大の証拠です。今回は期待リターン 5.35 万円が一致するか確認します。

Q1為替制約を「50 万円 → 60 万円」に緩めると、最適解はどう変化する？
A米国株が増え、期待リターンが上がる
B配分は変わらない
C現金が増える
Dすべての資産が均等になる
正解: A
為替制約が緩むと、リターン 8% の米国株を増やせます。元々制約 = 50 で米国 30 + 新興国 20 = 50 だったので、新興国は 20 万円上限に固定されたまま米国が伸びます。期待リターンは増加します。
Q2「what-if 分析」の目的として最も適切なのは？
Aコードのバグを探す
B前提条件が変わったとき最適解がどう変わるかを事前に把握する
CExcel を高速化する
D最適解を 1 つに絞り込む
正解: B
実務では「もし為替制約が緩んだら」「もし株価が下がったら」など複数シナリオを想定します。what-if 分析で事前に最適配分の変化を把握できます。
Q33 ツール（AI / Excel / Python）の使い分けとして適切なのは？
A常に Python が最良
B常に AI が最良
C小さく試すなら AI、構造把握なら Excel、繰り返し計算なら Python
Dどれも同じ
正解: C
AI は素早いラフ案、Excel は GUI で構造を見ながら触る、Python は大量の what-if やバージョン管理がしやすい。**3 つを役割で使い分ける**のが本講座の核心です。
Q4第 2-6 回で扱った 5 つの線形最適化応用に共通する構造は？
A必ず Python を使う
B常に最大化問題
C必ず変数が 5 個以上
D目的関数（最大化 or 最小化）+ 制約条件 + 決定変数
正解: D
生産計画・弁当・家具・輸送・ポートフォリオ — すべて「目的関数 + 制約 + 決定変数」の 3 点セットで定式化できます。題材が違っても**同じ枠組み**で考えられるのが線形最適化の強みです。
