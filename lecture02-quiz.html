Q1最適化問題の3要素として正しい組み合わせは？
A仮説・検証・結論
B利益・コスト・人数
C目的関数・制約条件・決定変数
D入力・処理・出力
正解: C
目的関数（何を最善にしたいか）、制約条件（守るべきルール）、決定変数（自分で決める量）の3つ。
Q2生産計画問題で「木材は18枚まで」にあたるのは3要素のうちどれ？
Aパラメータ
B目的関数
C決定変数
D制約条件
正解: D
「18枚まで」は守るべきルール（制約条件）。目的関数は「利益を最大にしたい」、決定変数は「机A・椅子Bの生産量」。
Q3次のうち「線形」でない式はどれ？
A3A + 4B ≤ 24
B1A + 2B
CA × B
D5A + B
正解: C
A × B は変数同士の掛け算で2次式。線形とは「すべてが1次式」であること。
Q4最大化問題と最小化問題の違いとして正しいのは？
A最小化問題には制約条件がない
B最小化問題では決定変数が不要
C方向が逆になるだけで、3要素の枠組みは同じ
D最大化問題はAIでは解けない
正解: C
この授業の例では最大化は制約≤（上限）、最小化は制約≥（下限）だが、常にそうとは限らない。重要なのは3要素の枠組みが同じであること。
Q5「バイト代5万円で生活する」例で、「合計は5万円以内」にあたるのは3要素のうちどれ？
A実行可能領域
B目的関数
C制約条件
D決定変数
正解: C
「合計5万円以内」「家賃は必ず払う」など守るべきルールは制約条件。決定変数は各項目への配分額、目的関数は総合満足度。
Q6線形最適化問題で、最適解は実行可能領域のどこに存在することが保証されている？
A必ず原点 (0,0)
B領域の中央
C領域の頂点（角）のどれか
D場所は決まっていない
正解: C
線形最適化では「最適解は実行可能領域の頂点（角）のどれかにある」ことが数学的に保証されている。だから無限の点を調べなくても、有限個の頂点だけを比較すれば最適解が見つかる。

Q1AIに最適化問題を伝えるとき、忘れがちな要素は？
A問題の種類
B非負制約（変数 ≥ 0）
C制約条件
D目的関数
正解: B
変数 ≥ 0（非負制約）は忘れがち。負の生産量は現実にはないが、明示しないとAIが負の値を出す可能性がある。
Q2AIに「利益を最大にして」とだけ伝えた場合、何が問題か？
A必ずエラーになる
BAIが理解できない
C制約条件や決定変数がわからず、正しく解けない
D問題ないのでそれで十分
正解: C
曖昧な指示では、AIが勝手に前提を置いて解いてしまう。目的関数・制約条件・決定変数・非負制約を構造化して伝えることが重要。
Q3AIの回答を検算するとき、チェックすべきことは？
A各制約に最適解を代入して不等式が成り立つか確認
Bグラフを描く
C目的関数の値だけ確認する
D別のAIに聞いてみる
正解: A
検算の3ステップ: ①制約チェック ②目的関数チェック ③非負チェック。まず制約を満たしているか確認。
Q4【検算】最適解 A = 3、B = 7 のとき、木材制約 2A + B の値と利益 3A + 4B の値はいくつ？（動画の検算パートの答え合わせ）
A2A+B = 18, 3A+4B = 37
B2A+B = 13, 3A+4B = 37
C2A+B = 13, 3A+4B = 28
D2A+B = 10, 3A+4B = 37
正解: B
2A + B = 2×3 + 7 = 13 ≤ 18（木材制約を満たす）。3A + 4B = 3×3 + 4×7 = 9 + 28 = 37（万円）。A, B はともに 0 以上なので非負制約もOK。検算の3ステップ（制約・目的関数・非負）がすべて通っています。
Q5「いきなり完成形のプロンプトを書け」と言われると戸惑う。動画で勧めている段階的な作り方の最初のステップは？
Aまず自分の言葉で現状を自然な日本語で書く
Bまず3つのAIに同じ質問を投げる
C最初から5要素を完璧に揃えた指示を書く
Dまず数式に置き換える
正解: A
段階1: 自分の言葉で書く → 段階2: AIに「何が足りない？」と聞く → 段階3: 5要素で構造化する。最初は雑でいい、というのが大事なポイント。
Q6同じプロンプトをChatGPT・Claude・Geminiに投げて答えがすべて一致した場合、何が言える？
A答えの信頼度が高いと判断できる
B何の情報も得られない
C検算は不要になる
D絶対に正しいことが証明される
正解: A
複数AIで答えが一致すれば信頼度は高い。ただし最終確認として、自分で検算の3ステップ（制約・目的関数・非負）を踏むのが安全。

Q1「最適解」と「最適値」の違いとして正しいのは？
A最適値＝制約条件の値
B最適解＝目的関数の値、最適値＝変数の値
C最適解と最適値は同じもの
D最適解＝変数の最善の組み合わせ、最適値＝目的関数の値
正解: D
最適解はA=3, B=7のような決定変数の組み合わせ。最適値は利益=37万円のような目的関数の値。
Q2組立時間の上限を24時間→30時間に増やすと利益が上がる可能性が高い。その理由として最も適切なのは？
A組立時間が有効制約（ボトルネック）で使い切っていたため、緩めると選べる範囲が広がるから
B木材の制約も同時に緩むから
C組立時間は元々余裕があったから
D机や椅子の利益率が上がるから
正解: A
利益が上がるのは、組立時間が有効制約（ぎりぎりまで使い切っている制約）だったから。ボトルネックを緩めると、机や椅子をもっと作れるようになり、利益が上がる。逆に、元々余っている制約（木材など）を緩めても何も変わらない。
Q3木材の上限を18枚→25枚に増やすと、利益はどうなる？
A増える
B変わらない
C予測できない
D減る
正解: B
木材は余裕がある制約（13/18枚）なので、上限を増やしても最適解は変わらない。
Q4「A≥5（机を最低5台作る）」という制約を追加すると？
A利益は同じか減る
B解がなくなる
C利益が増える
D利益が2倍になる
正解: A
制約を追加すると選択肢が狭まるため、利益は同じか減る。今回はA=3がA≥5を満たさないため、最適解が変わり利益が減る。
Q5製品B（椅子）の利益が4万円→6万円に上がった。最適解と利益はどう変わる？（各頂点で 3A + 6B を計算してから答えること）
A最適解の位置は (3, 7) のまま変わらず、利益だけが上がる
B椅子に集中する解（A=0, B=8）に変わり、利益が増える
C椅子の単価が上がるので利益は減る
D机だけを作る解になる
正解: A
各頂点で 3A + 6B を計算すると、(3, 7) = 51、(0, 8) = 48、(8, 2) = 36、(9, 0) = 27、(0, 0) = 0。最大は (3, 7) で 51 万円。利益は 37 → 51 で +14 万円。
気づき: 係数（単価）を変えても、制約構造が同じなら最適解の位置は同じ頂点に留まることがある（線形最適化の基本的な性質）。「予想と違って解の位置は動かない」というのがこのケースの学びどころ。
Q6「ボトルネック」とはどのような制約を指す？
A非負制約のこと
B限界まで使い切っている制約（有効制約）
Cまだ余裕がある制約
D数式の中で一番上に書かれた制約
正解: B
ボトルネック＝有効制約（限界まで使い切っている制約）。ここを緩めると最適値が改善する。経営的には「ボトルネックに投資するのが最も効果的」。

Q1食事問題で最小化したいのは？
A食品の数
Bカロリー
C栄養素
D食費（費用）
正解: D
食事問題の目的関数は「費用 = 130A + 160B → 最小化」。栄養素は制約条件。
Q2食事問題の制約条件の不等号の向きは？
A=（等号）
B≤（以下）
C問題によって異なる
D≥（以上）
正解: D
食事問題は「栄養素の必要量を満たす」ので≥（以上）。生産計画問題は「資源の上限」なので≤（以下）。
Q3「旅行プランで予算内に満足度を最大にしたい」場合、決定変数は？
A予算
B行く場所の選択
C満足度
D日程
正解: B
決定変数＝「自分で決められること」。旅行プランでは行く場所の選択。予算・日程は制約条件。
Q4食事問題の制約条件「炭水化物160g以上」を数式で表すと？（A=おにぎり 炭水化物40g、B=サラダ 炭水化物10g）
A40A + 10B = 160
B40A + 10B ≥ 160
C10A + 40B ≥ 160
D40A + 10B ≤ 160
正解: B
「炭水化物160g以上」→ 40A + 10B ≥ 160。最小化問題では制約が≥（下限）になることが多い。
Q5食事問題の解で「おにぎり 3.33個・サラダ 2.67個」と小数になった。動画ではこの扱いを何と呼んでいた？
A非有界
BWhat-if分析
C検算の3ステップ
D連続緩和
正解: D
線形最適化では変数を実数として扱う＝連続緩和。整数の答えが必要な場合は別途「整数計画法」で解く。連続緩和の答えは整数解の良い目安になる。
Q6食事問題でビタミンの必要量を 20mg → 30mg に増やすと、最小費用はどうなると予想できる？（ビタミンは元の最適解で有効制約だった）
A費用が減る
B変わらない
C解なしになる
D費用が増える
正解: D
最小化では「有効制約を厳しくすると目的関数が悪化（＝費用が増える）」。最大化のときの「有効制約を緩めると改善」と表裏一体の関係。
