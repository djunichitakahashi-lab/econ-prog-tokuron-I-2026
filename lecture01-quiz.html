Q1最適化問題の3要素として正しい組み合わせは？
A入力・処理・出力
B仮説・検証・結論
C目的関数・制約条件・決定変数
D収集・分析・解釈
正解: C
「何を良くしたいか（目的関数）」「守るべきルール（制約条件）」「自分で決められること（決定変数）」の3つ。
Q2アルバイトのシフト作成で「人件費を最小にしたい」にあたるのは？
A制約条件
B決定変数
C目的関数
Dパラメータ
正解: C
「何を良くしたいか」＝目的関数。人件費の最小化が目的。
Q3「各時間帯に最低2人は必要」はどの要素にあたるか？
A目的関数
B制約条件
C決定変数
D最適解
正解: B
「守らなければならないルール」＝制約条件。
Q4次のうち、最適化問題として捉えられるのはどれか？
A今日の天気を調べる
B予算5万円で満足度が最も高い旅行計画を立てる
C友達にメッセージを送る
D映画の感想を書く
正解: B
目的関数（満足度の最大化）、制約条件（予算5万円・日程）、決定変数（行き先・宿）の3要素がある。

①確認クイズ（全4問正解でSTEP 4解放）
Q1データ分析の5ステップの正しい順序は？
A分析 → 収集 → 可視化 → 整理 → 解釈
B収集 → 分析 → 整理 → 解釈 → 可視化
C収集 → 整理 → 分析 → 可視化 → 解釈
D整理 → 収集 → 可視化 → 分析 → 解釈
正解: C
まずデータを集め、きれいにし、特徴を見つけ、グラフにし、意味を読み取る。
Q2実務でデータ分析の時間の半分以上を占めると言われるステップは？
A収集
B整理
C分析
D可視化
正解: B
表記統一・欠損値処理・異常値除去など、データを分析できる形に整える作業は非常に手間がかかる。
Q32つの数値の関係（例：駅距離と価格）を見るのに適したグラフは？
A棒グラフ
B折れ線グラフ
C散布図
Dヒストグラム
正解: C
散布図は2つの数値の関係を点でプロットして視覚的に把握するグラフ。
Q4AIだけではデータ分析が不十分な理由として適切でないのは？
Aデータの品質を判断できない
B分析の前提の妥当性を確認しない
C文脈に応じた解釈ができない
D計算速度が遅い
正解: D
AIの計算速度は速い。問題は品質判断・前提確認・文脈に応じた解釈ができないこと。
🔒 4問すべて正解するとSTEP 4が解放されます。
